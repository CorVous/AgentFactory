# AFK trunk failure paths + launcher hardening

## Parent

[PRD-0001: Ralph-Loop mesh for AgentFactory V1](../PRD.md)

## What to build

The unhappy paths the slimmed #03 deferred. Once #03's tracer bullet is merged, this issue makes the system robust to invalid inputs and Foreman aborts. Hermetic / unit-testable; no live-model demo required.

### Launcher hardening

- Refuse to run unless invoked from inside a worktree of `<project>` (clear error message).
- Fail fast and clearly if `feature/<slug>` does not exist on the project, or if `.scratch/<slug>/issues/` is missing/empty on that branch.
- Otherwise: ensure `feature/<slug>` exists off `main` (creates if absent, fast-forwards if behind) before materialising the kanban worktree.

Each check has a focused unit test that exercises the launcher's pre-flight pure-function (separate the side-effects from the validation logic for testability).

### Foreman abort path

On test-fail or any abort signal (timeout, kill, exception in the Ralph Loop):

- Remove the `Claimed-by:` line from the issue file.
- Call `disposeWorktree` to remove the per-issue worktree.
- `git branch -D feature/<slug>-<NN>-<slug>` so stale refs do not accumulate (story #15).
- Exit non-zero so the Kanban logs the failure.
- Issue returns to ready (selectable on the next Kanban scan).

### Worktree-manager abort tests

Add tests to the existing #03 hermetic suite covering:

- Abort in the middle of a Ralph Loop → `disposeWorktree` removes the worktree and per-issue branch ref.
- Abort with uncommitted changes in the per-issue worktree → state is still cleaned up (don't leak the dirty tree).
- Abort after `reintegrate` started but before it finished → the partial merge is rolled back or never applied.

### Spawn-decision input-shape coverage

Extend the existing #03 spawn-decision unit tests with the cases #03 deferred:

- Blocked-issue input shape skipped (the actual depends-on logic lives in #07; this AC just exercises the contract).
- Issue file with malformed preamble (missing `Status:` line) skipped with a clear log line.
